# CartellaTombolaPDF

## Setting up
```
pip3 install jinja2
pip3 install pdfkit
sudo apt install wkhtmltopdf
```

## Run the script
```
cat cartelle_set.txt > python3 pdf_formatter.py
```

## _cartelle_set.txt_ sample
Input must be formatte in the following way: [cartelle_set sample] (cartelle_set.txt)
